# Positive Negative Zero

## Description

Checks whether a given input number is positive, negative, or zero and prints the corresponding result.

## Approach

Read a number from the user. Compare it to zero using conditional expressions. If the number is greater than zero, it is positive. If it is less than zero, it is negative. Otherwise, it is zero. Print the appropriate message.

## Algorithm

```
ALGORITHM POSITIVE_NEGATIVE_ZERO
// Classifies a number as positive, negative, or zero
// Input  : An integer num
// Output : A string indicating whether num is positive, negative, or zero

Read num from user
If num is greater than 0
    Set result = "number is positive"
Else if num is less than 0
    Set result = "number is negative"
Else
    Set result = "number is zero"
Print result
```

## Time Complexity

O(1) — a fixed number of comparisons are performed regardless of the input value.
