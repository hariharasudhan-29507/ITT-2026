# Largest of Three Numbers

## Description

Reads three numbers from the user and determines which of the three is the largest, printing the result.

## Approach

Compare the three numbers using conditional expressions. First check if the first number is greater than both others. If not, compare the second and third numbers to determine which is larger between them.

## Algorithm

```
ALGORITHM LARGEST_OF_THREE
// Finds and prints the largest among three input numbers
// Input  : Three integers a, b, and c
// Output : A message indicating which number is the largest

Read a, b, c from user
If a is greater than b AND a is greater than c
    Print "number 1 is largest"
Else if b is greater than c
    Print "number 2 is largest"
Else
    Print "number 3 is largest"
```

## Time Complexity

O(1) — a fixed number of comparisons are made regardless of input values.
